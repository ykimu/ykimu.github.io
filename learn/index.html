学習コースLearning course

研究の技術を、実践的に学ぶLearn research skills, hands-on

データ収集から各種の分析技術まで、「概要 → 理論 → 実際に動かせるコード」の順に学べる、全8モジュールの学習コースです。各モジュールは独立して読め、手を動かせるインタラクティブなデモと、実行できる Python の例をそなえています。高専生相当から、はじめて学ぶ一般の方までを対象にしています。

この学習コースは日本語のみで提供していますThis learning course is available in Japanese only.

コースの構成How the course works

3つの段階で学ぶ

この学習コースは、研究やデータ分析の現場で使う技術を、順を追って実践的に身につけることを目的としています。全体は大きく3つの段階に分かれています。

  • 第1段階：データを集める。分析の前に、まず素材となるデータが要ります。ソフトウェア開発の履歴からデータを掘り出す方法と、集めた生データを分析に使える形へ整える方法を学びます。
  • 第2段階：集めたデータを分析する。整ったデータから意味を引き出すための代表的な技術を、自然言語処理・ネットワーク分析・機械学習・深層学習・大規模言語モデル・プログラム解析に加え、統計分析・データ可視化・因果推論・時系列分析と、分野を横断して学びます。
  • 第3段階：技術をつないで実践する。第1・第2段階で学んだ技術を1本のパイプラインにつなぎ、公開データを使った現実的な課題を解く総合演習に取り組みます。各演習には、その場で実行できる Colab / Jupyter ノートブックがついています。

どのモジュールも、次の3つの層を積み上げる構成です。まず概要でその技術が何のためにあるのかをつかみ、次に仕組み（座学）で背後の理論・定義・式をていねいに理解し、最後にコードで動かすで Python の定番ライブラリやページ内デモを使って実際に確かめます。さらに各モジュールには、ブラウザ上で手を動かせるインタラクティブ・デモを用意しました。読みながら値を書き換え、結果の変化を確かめられます。

手を動かす題材には、実在の公開データセットや、ページ内の小さなサンプルだけを使います。特別な環境がなくても、読みながら試せる構成です。関心のあるモジュールから、どの順番で読んでもかまいません。

学習の進め方Learning roadmap

第1・第2段階で個々の技術を学び、第3段階の実践編でそれらを1本のパイプラインにつないで、現実的な課題を解きます。Stages 1 and 2 build up individual techniques; Stage 3 connects them into a single pipeline to solve realistic problems.

学び方How to study

手を動かしながら、自分のペースで

各モジュールは独立して読めます。興味のある技術から始めても、第1段階から順にたどってもかまいません。読みながらデモの数値を書き換え、コード例を手元で動かしてみると、理解がぐっと深まります。

これらの技術が研究の現場でどう使われているかは、研究や教育のページもあわせてご覧ください。

この学習コースについてAbout this course

「研究で使う技術を実践的に学ぶ」学習コースの入口ページです。データ収集から各種の分析技術まで、全8モジュールを2つの段階に分けて案内します。内容は分野標準の一般的なもので、特定の研究や論文には紐づきません。 The entry page of a learning course on research skills. It maps eight modules, grouped into two stages, from data collection to analysis techniques. The content is standard and general, not tied to any specific study.
